# Our mission
Currently existing Augmented Reality design applications — Apple Reality Composer, Adobe Aero, AR Makr, Torch AR, Assemblr — are limited in functionality and require external programs to be useful. They are inefficient and not made with focus on designing in 3d.  

With more artists and designers using iPads and Apple heavily investing in ARKit, this is the perfect time to create a next generation design tool. It will allow designers to create directly in the context of the real world; be it a chair, poster, new interior for a shop or an art piece.

# Our technology
We are building a native iOS application using [Xcode](https://developer.apple.com/xcode/), [Swift](https://developer.apple.com/swift/) and [ARKit](https://developer.apple.com/augmented-reality/arkit/). The app combines intuitive object manipulation, editing and drawing, all in Augmented Reality. 

This allows an artist, designer or anyone to start creating and finish the project directly in the app, without having to switch between multiple programs or even devices. We built an initial prototype and this proved intuitive to use.
